# lageco-frontend
LAGECO Frontend – Site web des Cours du Soir Bilingue LAGECO Ce projet a pour objectif de développer le site web officiel de l’établissement Cours du Soir Bilingue LAGECO, une école offrant des cours du soir, des cours à domicile, des formations en langues (français, anglais, allemand, espagnol) et en informatique.
